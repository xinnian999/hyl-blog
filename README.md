# 简介

本博客项目基于 create-react-app 搭建。

## 目录解构

| 目录       | 描述                     |
| ---------- | ------------------------ |
| assets     | 静态文件目录             |
| components | 通用组件                 |
| hooks      | 通用的自定义 react-hooks |
| jquery     | jquery 源码              |
| Layout     | 博客整体布局代码         |
| router     | 路由配置                 |
| store      | redux 配置               |
| style      | 公共样式                 |
| utils      | 公共工具函数             |
| views      | 所有路由模块代码         |

## 运行项目

克隆代码后，运行以下命令:

### `npm i`

安装依赖

### `npm run dev`

本地运行代码

### `npm run build`

打包代码

### `npm run deploy`

部署代码
